## Features

- **Stream** any YouTube Music track without an account
- **Download** songs in MP3/M4A format for offline listening
- **Subscribe** to artists and channels — local subscriptions, no login needed
- **Background Playback** — music continues while you use other apps
- **Modern Dark UI** — beautiful Material You design with moon-themed colors
- **No Ads** — completely ad-free experience
- **No Google account required**
- **Playlist support** — create and manage local playlists
- **Search** YouTube Music catalog
- **Mini Player** — floating controls in notification bar

```bash

cd youtube-music-cliant-andorid-apk
./gradlew assembleRelease
```

## Tech Stack

- **Language**: Kotlin
- **UI**: Material Design 3 + ViewBinding
- **YouTube Extractor**: NewPipe Extractor
- **Media Playback**: ExoPlayer / Media3
- **Download**: WorkManager
- **Database**: Room
- **Networking**: OkHttp + Retrofit
- **Image Loading**: Glide
